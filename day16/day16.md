# Day 16・17：振り返り・次のステップの展望

## 17日間の学習まとめ

### Phase 1（Day 1〜4）：ML基礎・テキストの数値化
線形回帰から始まり、テキストを数値に変換する方法を学んだ。BoW・TF-IDFで「文書らしさ」を表現し、ロジスティック回帰でIMDb 25,000件のレビュー分類（精度88%）を達成。

### Phase 2（Day 5〜9）：単語埋め込み・ニューラルネット
Word2Vecで「意味の近さ」を数値化。PyTorchで順伝播・逆伝播を手実装し、RNN・LSTMの限界からAttentionの必要性を理解。

### Phase 3（Day 10〜14）：Transformer・大規模モデル
位置エンコーディング・Multi-Head Attention・残差接続でTransformerの全体構造を理解。BERTで感情分析、GPTでテキスト生成、Helsinki-NLPで翻訳を実装。

### Phase 4（Day 15）：パイプライン
翻訳→要約のエンドツーエンドパイプラインを実装。max_position_embeddingsの制限を発見し、文章分割で解決。
